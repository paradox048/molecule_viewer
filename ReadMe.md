
## Executing program
- Compiling the program:
1. Run export LD_LIBRARY_PATH=$LD_LIBRARY_PATH: into the command line
```
> export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:
```
2. run make into the command line
```
> make
```
* Expected output:
```

clang mol.o -shared -o libmol.so
```
